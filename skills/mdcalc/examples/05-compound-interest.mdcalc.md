# 计算器规则

## 版本
0.1

## 功能简介
复利终值与各年末本息。本金 10000，年利率 5%，复利 10 年。年末本息 = 本金 × (1+利率)^年数。演示 `seq` 年份、`func`+`copy` 用 `POW` 与上一行递推两种方式对照。

## 使用流程
1. 用 edit_file 向"操作记录"中的 ops 数组追加一个动作对象
2. 用 bash 运行 `mdcalc <file>` 对文件求值
3. 用 read_file 读取"# 结果"部分的表格

# 操作记录

```js
const ops = [
  {op:'data', area:'A1', values:['年数']},
  {op:'data', area:'B1', values:['本息']},
  {op:'data', area:'C1', values:['较本金增长%']},
  {op:'seq',  area:'A2:A11', from:0, to:9, step:1},
  {op:'func', area:'B2',     values:['ROUND(10000 * POW(1.05, A2), 2)']},
  {op:'copy', from:'B2',     to:'B3:B11'},
  {op:'func', area:'C2',     values:['ROUND((B2 - 10000) / 10000 * 100, 2)']},
  {op:'copy', from:'C2',     to:'C3:C11'}
]
```
