# 计算器规则

## 版本
0.1

## 功能简介
消费贷款等额本息还款计划表。贷款本金 5.5 万元，期限 12 期（每月 1 期，共 1 年），年利率 7.2%（月利率 0.006）。每期月供固定（等额本息：月供 = 本金 × 月利率 × (1+月利率)^期数 / ((1+月利率)^期数 − 1)），其中利息逐期递减、本金逐期递增。

第一行（第 1 行）为表头（文本），数据从第 2 行起共 12 行：求值器检测到第 1 行每列都是文本时，会把它当作列名渲染、数据行从第 2 行开始并以 1.. 起编行号。

本表用 `seq` 与 `copy` 两个 op 构造，操作记录紧凑：
- 表头 A1:F1 一次 `data` 写入 6 个列名
- A 期次 1..12：`seq` from:1 to:12 step:1（写入 A2:A13）
- E 月供合计（固定）：E2 = 本金 × 月利率 × (1+月利率)^期数 / ((1+月利率)^期数 − 1)，用 `POW` 求值；再 `copy` from:'E2' to:'E3:E13'（每期月供相同）
- B2 期初本金 = 55000；B3..B13 = 上期期末本金：先写 B3=`ROUND(F2,2)`，再 `copy` from:'B3' to:'B4:B13'
- D 月还利息 = ROUND(B × 0.006, 2)：先写 D2=`ROUND(B2 * 0.006, 2)`，再 `copy` from:'D2' to:'D3:D13'
- C 月还本金 = ROUND(E − D, 2)（月供固定，本金 = 月供 − 利息，逐期递增）：先写 C2=`ROUND(E2 - D2, 2)`，再 `copy` from:'C2' to:'C3:C13'
- F 期末本金 = ROUND(B − C, 2)：先写 F2=`ROUND(B2 - C2, 2)`，再 `copy` from:'F2' to:'F3:F13'

每个公式都内嵌 `ROUND(..., 2)`，使舍入后的值既用于显示、又参与后续计算，避免"显示舍入、内部未舍入"的数据不一致。最后一期期末本金应为 0（在舍入精度内）。

列含义（表头）：
- A 期次（1..12）
- B 期初剩余本金（该期还款前的本金余额）
- C 月还本金（逐期递增 = 月供 − 月还利息）
- D 月还利息（逐期递减 = 期初剩余本金 × 月利率）
- E 月供合计（固定 = 等额本息月供）
- F 期末剩余本金（= 期初剩余本金 − 月还本金）

## 使用流程
1. 用 edit_file 向"操作记录"中的 ops 数组追加一个动作对象
2. 用 bash 运行 `mdcalc <file>` 对文件求值
3. 用 read_file 读取"# 结果"部分的表格

# 操作记录

```js
const ops = [
  {op:'data', area:'A1:F1',  values:['期次','期初本金','月还本金','月还利息','月供合计','期末本金']},
  {op:'seq',  area:'A2:A13',  from:1, to:12, step:1},
  {op:'func', area:'E2',      values:['ROUND(55000 * 0.006 * POW(1.006, 12) / (POW(1.006, 12) - 1), 2)']},
  {op:'copy', from:'E2',      to:'E3:E13'},
  {op:'func', area:'B2',      values:['55000']},
  {op:'func', area:'B3',      values:['ROUND(F2, 2)']},
  {op:'copy', from:'B3',      to:'B4:B13'},
  {op:'func', area:'D2',      values:['ROUND(B2 * 0.006, 2)']},
  {op:'copy', from:'D2',      to:'D3:D13'},
  {op:'func', area:'C2',      values:['ROUND(E2 - D2, 2)']},
  {op:'copy', from:'C2',      to:'C3:C13'},
  {op:'func', area:'F2',      values:['ROUND(B2 - C2, 2)']},
  {op:'copy', from:'F2',      to:'F3:F13'}
]
```
